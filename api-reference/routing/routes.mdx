## List Routes

```bash GET /fwclouds/:fwcloud/firewalls/:firewall/routingTables/:routingTable/routes

  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

## Create Route

<CodeGroup>
```bash POST /fwclouds/:fwcloud/firewalls/:firewall/routingTables/:routingTable/routes
curl -X POST "https://api.example.com/fwclouds/1/firewalls/1/routingTables/1/routes" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "10.0.0.0/24",
    "gateway": "192.168.1.1",
    "interface": "eth0",
    "metric": 100,
    "comment": "Route to internal network",
    "active": true
  }'
```
</CodeGroup>

**Request Body**
- `to` (string, optional) - Destination network (CIDR notation)
- `gateway` (string, optional) - Gateway IP address
- `interface` (string, optional) - Network interface
- `metric` (integer, optional) - Route metric/priority
- `comment` (string, optional) - Description
- `active` (boolean, optional) - Whether route is enabled

## Copy Routes

<CodeGroup>
```bash POST /fwclouds/:fwcloud/firewalls/:firewall/routingTables/:routingTable/routes/copy
curl -X POST "https://api.example.com/fwclouds/1/firewalls/1/routingTables/1/routes/copy" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "routes": ["1", "2"],
    "to": 5,
    "offset": "below"
  }'
```
</CodeGroup>

Copies routes to a new position.

**Request Body**
- `routes` (array, required) - Route IDs to copy
- `to` (integer, required) - Target route ID to position relative to
- `offset` (string, required) - Position: "above" or "below"
