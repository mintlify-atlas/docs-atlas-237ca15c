## Create HAProxy Rule

<CodeGroup>
```bash POST /fwclouds/:fwcloud/firewalls/:firewall/system/haproxyRules
curl -X POST "https://api.example.com/fwclouds/1/firewalls/1/system/haproxyRules" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "frontend_ip": "0.0.0.0",
    "frontend_port": 80,
    "backend_servers": [
      {"ip": "192.168.1.10", "port": 8080},
      {"ip": "192.168.1.11", "port": 8080}
    ],
    "mode": "http",
    "balance": "roundrobin",
    "comment": "HTTP load balancer",
    "active": true
  }'
```
</CodeGroup>

**Request Body**
- `frontend_ip` (string, optional) - Frontend bind IP address
- `frontend_port` (integer, optional) - Frontend listen port
- `backend_servers` (array, optional) - Backend server configurations
- `mode` (string, optional) - Protocol mode: "http" or "tcp"
- `balance` (string, optional) - Load balancing algorithm: "roundrobin", "leastconn", "source"
- `comment` (string, optional) - Description
- `active` (boolean, optional) - Whether rule is enabled
- `group` (integer, optional) - HAProxy group ID

## Install HAProxy Configuration

<CodeGroup>
```bash PUT /fwclouds/:fwcloud/firewalls/:firewall/system/haproxyRules/install
curl -X PUT "https://api.example.com/fwclouds/1/firewalls/1/system/haproxyRules/install" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Compiles and installs HAProxy configuration to the firewall.

This endpoint:
1. Compiles all HAProxy rules into haproxy.cfg
2. Uploads the configuration to `/etc/haproxy/haproxy.cfg`
3. Validates the configuration
4. Emits progress events via WebSocket

**Response Events (WebSocket)**
```json
{
  "status": "start",
  "message": "Installing HAProxy rules"
}
```
