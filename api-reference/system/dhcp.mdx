## List DHCP Rules

<CodeGroup>

## Create DHCP Rule

<CodeGroup>
```bash POST /fwclouds/:fwcloud/firewalls/:firewall/system/dhcpRules
curl -X POST "https://api.example.com/fwclouds/1/firewalls/1/system/dhcpRules" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "interface": "eth0",
    "range": "192.168.1.100-192.168.1.200",
    "router": "192.168.1.1",
    "netmask": "255.255.255.0",
    "dns": ["8.8.8.8", "8.8.4.4"],
    "max_lease_time": 86400,
    "default_lease_time": 43200,
    "comment": "DHCP pool for LAN",
    "active": true
  }'
```
</CodeGroup>

**Request Body**
- `interface` (string, optional) - Network interface
- `range` (string, optional) - DHCP range (format: "start-end")
- `router` (string, optional) - Default gateway
- `netmask` (string, optional) - Subnet mask
- `dns` (array, optional) - DNS servers
- `max_lease_time` (integer, optional) - Maximum lease time in seconds
- `default_lease_time` (integer, optional) - Default lease time in seconds
- `comment` (string, optional) - Description
- `active` (boolean, optional) - Whether rule is enabled
- `group` (integer, optional) - DHCP group ID

## Install DHCP Configuration

<CodeGroup>
```bash PUT /fwclouds/:fwcloud/firewalls/:firewall/system/dhcpRules/install
curl -X PUT "https://api.example.com/fwclouds/1/firewalls/1/system/dhcpRules/install" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</CodeGroup>

Compiles and installs DHCP configuration to the firewall.

This endpoint:
1. Compiles all DHCP rules into dhcpd.conf
2. Uploads the configuration to `/etc/dhcp/dhcpd.conf`
3. Validates the configuration
4. Emits progress events via WebSocket
