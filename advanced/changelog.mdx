## Recent Releases

### [2.3.0] - 2026-01-22

#### Added
- **VyOS Support**: Complete policy compiler for VyOS firewall platform
  - VyOS-specific script templates (header and footer)
  - API validation and compiler detection for VyOS
- **Dangerous Rules Detection**: Identifies potentially risky firewall rules during compilation
  - Warning/critical flags for dangerous rules
  - Per-rule progress updates
  - Configuration loaded from JSON with database persistence
- **Enhanced Rule Management**:
  - `firewallApplyToId` support for DHCP, Keepalived, and HAProxy rules
  - Cluster-wide rule handling
- **WebSocket Improvements**: Room management for FWCloud locks with previous lock owner information

#### Changed
- **Express 5 Migration**: Updated routes and E2E tests for Express 5
- **Request Parameters**: Now accepts bracketed array syntax
- **OpenVPN**: Configurations marked dirty on any option change
- **Group Name Handling**: `sanitizeGroupName` handles null input and strips leading `!` characters
- **Dependencies**: Updated npm dependencies; mocha pinned to 11.3.0

#### Fixed
- System services apply migration
- Dangerous rules checks and loading
- CLI startup with ESM yargs
- API call for forcing FWCloud locks

### [2.2.0] - 2025-09-30

#### Added
- VyOS policy compiler and templates
- Force lock API call for FWClouds

#### Fixed
- Certificate name validation (no non-alphanumeric starting characters)

### [2.1.5] - 2025-09-17

#### Fixed

### [2.1.4] - 2025-09-11

## Major Version: 2.1.0

### [2.1.0] - 2025-09-04

This was a significant feature release:

#### Added
- **Exclusive FWCloud Access**: Lock mechanism for concurrent editing
  - Only one user can edit a FWCloud at a time
  - Read-only mode for additional users
- **AI Integration**: Query firewall policies using AI
  - Integration with AI API (OpenAI)
  - Natural language policy queries
  - Configuration section for AI settings
- **Multi-VPN Support**:
  - WireGuard VPN management
  - IPsec (strongSwan) VPN management
  - Plugins for WireGuard and IPsec
  - Systemctl management for VPN services
- **Database Migrations**: Schema updates for new features
- **Test Coverage**: Comprehensive tests for new features

#### Fixed
- Multiple bug fixes for stability and performance
- Firewall mark modification bug
- Updated Node.js packages

## Upgrade Guides

### From 2.x to Latest

```bash
# Package installation
sudo dpkg -i fwcloud-api_2.3.1.deb  # or .rpm

# Manual installation
npm run stop
git pull
npm install
npm run build
node fwcli migration:run
npm start
```

### From 1.x to 2.x

Major version upgrade requires careful migration:

1. Backup database:
   ```bash
   mysqldump -u fwcloud_user -p fwcloud > backup_pre_v2.sql
   ```

2. Update application:
   ```bash
   npm run update
   ```

3. Verify FWCloud Agent compatibility

4. Test all integrations
