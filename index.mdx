<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with FWCloud API in minutes</p>
  
  <Steps>
    <Step title="Install and start the API">
      Clone the repository and install dependencies:
      
      ```bash
      git clone https://github.com/soltecsis/fwcloud-api.git
      cd fwcloud-api
      npm install
      npm run build
      npm start
      ```
      
      The API will start on the configured port (default: 3131).
    </Step>
    
    <Step title="Configure database connection">
      Copy the example environment file and configure your database:
      
      ```bash
      cp .env.example .env
      ```
      
      Edit `.env` to set your MySQL connection details. Then run migrations:
      
      ```bash
      node fwcli migration:run
      ```
    </Step>
    
    <Step title="Make your first API call">
      Authenticate and create your first FWCloud:
      
      ```bash
      curl -X POST https://your-api-host/fwclouds \
        -H "Content-Type: application/json" \
        -H "Cookie: session=your-session-cookie" \
        -d '{"name": "My First FWCloud"}'
      ```
      
      <Accordion title="Example Response">
        ```json
        {
          "id": 1,
          "name": "My First FWCloud",
          "created_at": "2024-03-04T10:00:00.000Z",
          "updated_at": "2024-03-04T10:00:00.000Z"
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
